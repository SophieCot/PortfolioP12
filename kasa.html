Kasa

Contexte et Objectifs

Contexte : Kasa, entreprise spécialisée dans la location d’appartements entre particuliers, entreprend une refonte totale de son site. Initialement développé en ASP.NET, le site passe désormais à une stack JavaScript moderne avec React pour le front-end et Node.js pour le back-end. Objectifs : Développer l’application React en respectant les maquettes Figma fournies. Mettre en place une navigation fluide avec React Router. Intégrer un design responsive et une interface utilisateur ergonomique. Développer des composants réutilisables en React. Implémenter un système de gestion des annonces à partir d’un fichier JSON temporaire.

Fonctionnalités Principales

  • Galerie d’images Défilement infini des photos (retour à la première image après la dernière). Suppression des boutons de navigation si une seule image est présente. Maintien d’une hauteur fixe des images, avec centrage automatique.
  • Menu déroulant (Collapse) Initialisation en mode fermé. Ouverture/Fermeture via un clic utilisateur. Animation fluide pour une transition agréable.
  • Navigation et Affichage React Router pour la gestion des routes. Affichage dynamique des annonces en fonction du fichier JSON. Composants réutilisables pour optimiser la maintenance du code.

Outils et Technologies

Les principaux outils et technologies utilisés dans ce projet sont :

  • React.js : Développement de l’application en mode composant.
  • React Router : Gestion des pages et navigation fluide.
  • SASS (SCSS) : Gestion avancée des styles, modularisation du CSS.
  • Animations CSS : Pour les transitions de menus et effets interactifs.
  • JSON : Utilisation temporaire d’un fichier JSON comme base de données.
  • Vite.js : Outil rapide pour la création et le développement d’applications React.
  • Git/GitHub : Versionning et suivi des évolutions du projet.
