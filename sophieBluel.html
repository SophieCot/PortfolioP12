Contexte et Objectifs

L’agence ArchiWebos travaille sur la conception du site portfolio d’une architecte d’intérieur. Ce site permettra de présenter ses réalisations et de gérer facilement ses contenus. En tant que développeur front-end, votre mission est de renforcer l’équipe et d’intégrer les nouvelles fonctionnalités en respectant le design et les spécifications techniques. L’objectif est de fournir une expérience utilisateur fluide et professionnelle, en s’appuyant sur une interface intuitive et un back-end performant.

Fonctionnalités Principales

Le projet comprend trois fonctionnalités clés à développer :

  • Page de présentation des travaux Intégration de la page à partir du HTML statique existant. Mise en forme des projets avec une galerie interactive.
  • Page de connexion de l’administrateur Création d’une interface sécurisée pour permettre au client de se connecter. Gestion des erreurs d’identification et des restrictions d’accès
  • Modale d’upload de médias Développement d’une fenêtre modale permettant d’ajouter de nouveaux visuels. Validation et prévisualisation des fichiers avant l’envoi. Communication avec le back-end pour stocker les médias.

Outils et Technologies

Les principaux outils et technologies utilisés dans ce projet sont :

  • HTML / CSS (SASS) : Pour l’intégration et le design responsive.
  • JavaScript : Gestion des interactions et de la logique client.
  • API REST : Communication avec le back-end pour l’authentification et la gestion des médias.
  • Git / GitHub : Suivi des versions et collaboration via le Kanban.
