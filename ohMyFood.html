Oh my food

Contexte et Objectifs

Contexte : OhMyFood est une startup innovante qui souhaite révolutionner l’expérience des restaurants gastronomiques en proposant un service de précommande en ligne. Déjà présente à New York, l’entreprise veut maintenant s’implanter à Paris avec un site web qui permet aux clients de composer leur menu à l’avance. Objectifs : Développer un site web responsive en mobile first. Répertorier les menus de 4 restaurants partenaires. Permettre aux utilisateurs de précommander leurs plats avant leur arrivée au restaurant. Utiliser SASS pour structurer le code CSS efficacement. Intégrer des animations CSS pour dynamiser l’expérience utilisateur.

Fonctionnalités Principales

  • Expérience Utilisateur (UX/UI) Navigation fluide et responsive (mobile first, adaptable aux tablettes et desktop). Animations CSS pour améliorer l’interactivité du site (chargement des plats, effets de transition). Design moderne et intuitif, conforme aux maquettes Figma fournies.
  • Parcours Utilisateur Consulter la liste des restaurants partenaires. Sélectionner un restaurant pour voir son menu détaillé. Composer un menu personnalisé en ajoutant des plats. Finaliser la réservation avec les plats sélectionnés.

Outils et Technologies

Les principaux outils et technologies utilisés dans ce projet sont :

  • HTML5 / CSS3 : Structure et stylisation du site.
  • SASS (SCSS) : Découpage et organisation du CSS pour plus de clarté et de modularité.
  • Animations CSS : Transitions et effets pour améliorer l’expérience utilisateur.
  • Approche Mobile First : Priorisation de l’expérience utilisateur sur smartphone avant l’adaptation aux autres écrans.
